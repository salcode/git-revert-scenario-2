Git Revert Scenario 2

Git revert is a powerful tool to remove changes in a Git repository while moving forward (i.e. by NOT rewriting history).

Git Revert vs Reset

While git revert removes changes by moving forward in Git history, git reset removes changes by rewriting history, see https://git-scm.com/docs/git-reset.

Use the Right Tool for the Job

Both git revert and git reset are powerful commands for managing your Git repository. Which one I reach for depends on the situation.
